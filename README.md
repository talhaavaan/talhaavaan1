# Talha Avaan — Portfolio v2

Rebuilt with the Michael Smith template structure + Talha's real content.

## Stack
- React 19 + Vite 8
- Tailwind CSS v4
- Framer Motion (page transitions, whileInView)
- GSAP (hero entrance, footer marquee)
- hls.js (HLS video streaming in hero + footer)
- Instrument Serif + Inter fonts

## Setup

```bash
npm install
npm run dev
```

## Sections
1. **LoadingScreen** — Counter 000→100, cycling words (Edit/Color/Create/Motion), progress bar
2. **Hero** — HLS video bg, floating pill navbar (TA logo ring), name reveal, role cycling, CTA buttons, scroll indicator
3. **Trust** — Animated stats counters
4. **About** — Timeline + skills tags
5. **Services** — 6 service cards grid
6. **Portfolio** — 12-col bento grid with filters, hover reveal pills
7. **Showreel** — Particle canvas bg, 16:9 player, journal-style project rows
8. **Process** — 4-step workflow
9. **Testimonials** — Auto-scrolling marquee row
10. **Software** — Skill bars with intersection observer
11. **Why** — 6 reason cards
12. **Contact** — Split layout form
13. **Footer** — Flipped HLS video, GSAP marquee, social links, availability dot
